This is a test device I created as my first project after obtaining a Max for Live development environment.
Proper operation is not guaranteed.

Please note that this was created on a Mac.
This was made purely as an experimental test, so it hasn’t been thoroughly tested.
I’m also not sure whether this file will load correctly on Windows, so please understand these limitations.

Trigger – Starts recording into the buffer and simultaneously plays back the stretched audio.
The buffer has a limited length. It is not recommended to keep the Trigger turned on continuously.

Scan – Sets the amount of stretching (smaller values result in more stretching).

GSize – Sets the grain size (larger values produce a coarser sound).

PBSpeed – Sets the pitch (smaller values result in a lower pitch).

GSize - グレインのサイズです(値が大きいほど粗くなる)
PBSpeed - 音高の高さを設定します(値が小さいほど音が低くなる)
